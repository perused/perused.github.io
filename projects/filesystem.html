Virtual File System

Description here...

In this project, I implemented a virtual file system in C. First, methods were created that simulated real file operations on a real disk such as read, write and delete. Then, a merkel tree hash was applied to each read and write to verify the file system had not been corrupted at any point. After this was done, the system was extended to utilise the Filesystem in Userspace (FUSE) Linux module. This resulted in the file system operating like an actual external drive. Other extensions included repacking the file system in order to ensure disk space could be fully utilised as well as making each of the methods thread safe in order to allow for multi-threaded functionality in the future.
